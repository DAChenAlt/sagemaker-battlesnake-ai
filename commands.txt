# To recreate Lambda package

From a Linux machine:

```
virtualenv venv
source venv/bin/activate
pip install mxnet

cp SnakeSource/* venv/lib/python2.7/site-packages

cd venv/lib/python2.7/site-packages && zip -r ../../../../ lambda-ai-endpoint.zip . && cd -
```

aws s3 cp lambda-ai-endpoint.zip s3://battlesnake-aws-ca-central-1/cloudformation/ --profile prototype
aws s3 cp demo-sagemaker.yaml s3://battlesnake-aws-ca-central-1/cloudformation/deploy-battlesnake-endpoint.yaml --profile prototype
